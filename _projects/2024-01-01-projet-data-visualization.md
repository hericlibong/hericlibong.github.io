## Description du projet

Ce projet présente une analyse interactive des tendances de données pour l'année 2023. Utilisant des technologies modernes de visualisation, ce tableau de bord permet d'explorer les données sous différents angles.

## Technologies utilisées

- D3.js pour les visualisations interactives
- Tableau pour les graphiques statiques
- Python pour le prétraitement des données

## Résultats

Le projet a permis de mettre en évidence plusieurs tendances clés et a été présenté lors de la conférence DataViz 2023.
